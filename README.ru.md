# Klaxon

[English](README.md) · **Русский**

**Полноэкранные напоминания о встречах для macOS, которые невозможно пропустить.**

[![Скачать Klaxon](https://img.shields.io/badge/%E2%AC%87%EF%B8%8E%20Скачать-Klaxon.dmg-cc2936?style=for-the-badge&logo=apple&logoColor=white)](https://github.com/sleonia/klaxon/releases/latest/download/Klaxon.dmg)

Klaxon — это лёгкое приложение в строке меню, которое разворачивает
полноэкранное напоминание на каждом дисплее прямо перед началом встречи из
календаря, чтобы вы её действительно заметили. Если у встречи есть ссылка на
видеозвонок, одно нажатие открывает его сразу.

Сделано для тех, кто теряет счёт времени: с головой ушёл в работу, весь день
переключается между задачами или просто устал от крошечных уведомлений,
которые исчезают, не успеешь поднять глаза.

## Возможности

- **Полноэкранные напоминания** на каждом подключённом дисплее, поверх других
  приложений и рабочих столов (Spaces), с настраиваемым временем показа (в
  момент начала, за 1/2/5/10/15 минут).
- **Управление с клавиатуры** — **↵** присоединяет к звонку, **⌘1 / ⌘2 / ⌘3**
  откладывают, **esc** закрывает; на каждой кнопке видна её горячая клавиша.
- **Присоединение в одно нажатие** — распознаёт ссылки на 30+ сервисов (Zoom,
  Google Meet, Microsoft Teams, Webex, Whereby, Jitsi и другие) в URL, месте или
  заметках события и показывает кнопку Join для того, что нашёл.
- **Настраиваемый Snooze** — от 0 до 3 кнопок отсрочки на ваш выбор (по
  умолчанию 1 / 3 / 5 минут) плюс Dismiss.
- **Темы и свой фон** — десять встроенных градиентных тем, вариант **Random**,
  который выбирает новую тему для каждого напоминания, или ваше собственное
  изображение в качестве фона.
- **Звуки** — проигрывайте вместе с напоминанием любой системный звук macOS.
  Также есть возможность выключить любые звуки.
- **Работа с календарём** — поддерживает все учётные записи, уже настроенные в
  macOS Calendar (iCloud, Google, Microsoft/Exchange, …) через EventKit.
  Выбирайте, какие календари учитывать; при желании включайте события на весь
  день и те, от которых вы отказались.
- **Пауза откуда угодно** — глобальная горячая клавиша ⌃⌥P (и переключатель в
  строке меню) ставят напоминания на паузу и снова включают их.
- **Живёт в строке меню** — без иконки в Dock; показывает ближайшую встречу и
  таймер до неё или только иконку, если включить **Show icon only in menu bar**.
- **Запуск при входе** через `SMAppService`.
- **Надёжное планирование** — учитывает выход из сна и перевод часов, а перед
  показом делает повторную проверку, чтобы перенесённые или отменённые встречи
  не подняли устаревшее напоминание.

## Настройки

Откройте **Settings…** через иконку в строке меню (или нажмите **⌘,**). Есть две
вкладки:

**General**

- **Show alert** — за сколько до встречи появляется напоминание: в момент начала
  или за 1 / 2 / 5 / 10 / 15 минут.
- **Theme** — одна из десяти градиентных тем, **Random** (новый градиент на
  каждое напоминание) или **Custom Image…**, чтобы использовать своё изображение
  как фон.
- **Sound** — любой системный звук macOS или None, с кнопкой предпрослушивания.
- **Snooze buttons** — добавьте до трёх (по 1–120 минут каждая); они становятся
  кнопками **⌘1 / ⌘2 / ⌘3** на напоминании. Удалите все, чтобы полностью убрать
  отсрочку.
- **Pause all alerts** — тот же переключатель, что и глобальная горячая клавиша
  **⌃⌥P**.
- **Show icon only in menu bar** — скрывает название ближайшей встречи и таймер,
  оставляя только иконку. «Paused» всё равно отображается, так как это заданное
  вами состояние приложения.
- **Launch at login** — запускать Klaxon автоматически (доступно при запуске из
  установленного бандла приложения).
- **Test Alert…** — показать полноэкранное напоминание прямо сейчас для
  проверки.

**Calendars**

- Выберите, за какими именно календарями следит Klaxon — каждый показан со своим
  цветом и учётной записью.
- При желании включите **события на весь день** и **события, от которых вы
  отказались**.

## Установка

Требуется **macOS 14 (Sonoma) или новее**.

### Скачать

[**⬇︎ Скачать Klaxon.dmg**](https://github.com/sleonia/klaxon/releases/latest/download/Klaxon.dmg)
— всегда последняя версия. (Старые сборки — на странице
[Releases](https://github.com/sleonia/klaxon/releases).) Откройте DMG и
перетащите **Klaxon** в **Applications**.

### Первый запуск

Klaxon подписан только *ad-hoc*-подписью и **не нотаризован** — он не участвует
в платной программе Apple Developer. Поэтому при первом открытии скачанной
копии macOS Gatekeeper заблокирует её («не удаётся открыть, так как не удалось
проверить разработчика»). Обойдите это один раз через **правый клик → Open**,
затем **Open** в диалоге — или выполните:

```sh
xattr -dr com.apple.quarantine /Applications/Klaxon.app
```

После этого приложение запускается как обычно.

### Сборка из исходников

Нужны инструменты командной строки Xcode (`xcode-select --install`). Затем:

```sh
git clone https://github.com/sleonia/klaxon.git
cd klaxon
./Scripts/build-app.sh
```

Этот единственный скрипт собирает приложение, подписывает его и устанавливает
`Klaxon.app` в папку Applications. Откройте его, дайте доступ к календарю по
запросу — и в строке меню появится горн. Готово.

> Сборка подписана локально, поэтому при первом запуске macOS может попросить
> подтверждение. Если запуск заблокирован, щёлкните по приложению правой кнопкой
> и выберите **Open**.

> **Часто пересобираете?** По умолчанию каждая пересборка получает новую ad-hoc
> подпись, поэтому macOS каждый раз заново спрашивает доступ к календарю.
> Запустите `./Scripts/setup-signing.sh` **один раз**, чтобы создать постоянную
> локальную подпись — после этого пересборки сохраняют ту же подпись, и
> разрешение не сбрасывается.

Запустить набор тестов: `swift test`.

### Удаление

```sh
./Scripts/clean.sh
```

Удаляет установленное приложение, локальные артефакты сборки, сохранённые
настройки и выданное Klaxon разрешение на доступ к календарю.

## Разрешения

Klaxon нужен **полный доступ к календарю** (Full Calendar Access), чтобы знать,
когда начинаются встречи. Этот доступ используется только для локального чтения
времени событий и ссылок на встречи — ничего не покидает вашу машину. Доступ к
Универсальному доступу (Accessibility) не требуется.
